## Getting Started

### Account Registration
1. Visit the application website
2. Click on "Sign Up" or "Register"
3. Enter your username, email, and password
4. Verify your email if required
5. Log in to start using the app

### Login
1. Enter your username/email and password
2. Click "Login"
3. You'll be directed to your task dashboard

## Creating Tasks

### Basic Task Creation
1. Click the "+" button or "Add Task" button
2. Fill in the task title (required)
3. Add a description (optional)
4. Set the priority (Low, Medium, High)
5. Add tags to categorize your task
6. Set a due date if applicable
7. Click "Create Task"

### Task Fields Explained
- **Title**: Brief description of the task (1-255 characters)
- **Description**: More detailed information about the task
- **Priority**: Importance level (Low, Medium, High)
- **Tags**: Categories to organize tasks (e.g., work, personal, home)
- **Due Date**: Deadline for the task
- **Status**: Active or Completed (automatically set to Active)

## Managing Tasks

### Viewing Tasks
- Tasks are displayed in a card format
- Each card shows title, description, priority, due date, and tags
- Tasks are sorted by default with most recent at the top

### Editing Tasks
1. Click the "Edit" button on the task card
2. Modify any fields as needed
3. Click "Update Task" to save changes

### Marking Tasks Complete
- Click the checkbox icon next to the task
- Completed tasks will be visually marked and moved to the completed section
- To reactivate, click the checkbox again

### Deleting Tasks
1. Click the "Delete" button on the task card
2. Confirm deletion in the popup dialog
3. The task will be permanently removed

## Organizing Tasks

### Using Tags
1. Create tags by going to the "Tags" section
2. Add tags like "work", "personal", "shopping"
3. Assign tags to tasks during creation or editing
4. Filter tasks by tags to see related items

### Setting Priorities
- Low: Tasks that can wait
- Medium: Important but not urgent
- High: Urgent and important tasks

### Using Status
- Active: Tasks you're currently working on
- Completed: Finished tasks (can be reactivated)

## Searching and Filtering

### Search
- Use the search bar at the top to find tasks by title or description
- Search is case-insensitive
- Results update in real-time

### Filters
- **Status Filter**: Show only active or completed tasks
- **Priority Filter**: Show tasks of specific priority levels
- **Date Range**: Show tasks due today, this week, this month, or overdue
- **Tags**: Filter by specific tags

### Sorting
- **Due Date**: Sort by upcoming deadlines
- **Priority**: Sort by importance
- **Alphabetical**: Sort by title A-Z or Z-A
- **Date Created**: Sort by when tasks were added

## Recurring Tasks

### Creating Recurring Tasks
1. When creating a task, enable the "Recurring Task" option
2. Select the pattern type:
   - Daily: Repeats every day
   - Weekly: Repeats every week
   - Monthly: Repeats every month
   - Yearly: Repeats every year
3. Set the interval (e.g., every 2 days, every 3 weeks)
4. Optionally set an end date or occurrence limit
5. Save the task

### Managing Recurring Tasks
- Each occurrence is treated as a separate task
- When you complete an occurrence, the next one is automatically created
- Editing the original task affects future occurrences
- Deleting the original task stops future occurrences

## Due Dates and Reminders

### Setting Due Dates
1. When creating or editing a task
2. Click on the due date field
3. Select the date and time
4. Save the task

### Enabling Reminders
1. When setting a due date, enable the "Enable Reminder" option
2. Choose when you want to be reminded:
   - 1 hour before
   - 1 day before
   - 1 week before
   - Custom time

### Managing Notifications
- Allow browser notifications when prompted
- Notifications appear in your system tray
- You can snooze or dismiss notifications
- Manage notification settings in your browser settings

## Troubleshooting

### Common Issues

#### Can't Log In
- Check your username/email and password
- Ensure Caps Lock is off
- Reset your password if needed

#### Tasks Not Saving
- Check your internet connection
- Refresh the page and try again
- Contact support if the issue persists

#### Notifications Not Working
- Check browser notification permissions
- Ensure you've allowed notifications for the site
- Check your system notification settings

#### Search Not Finding Tasks
- Ensure you're spelling the search term correctly
- Search looks in both title and description
- Try using fewer keywords
