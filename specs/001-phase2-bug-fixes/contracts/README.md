# API Contracts

This directory contains the API contracts for the Phase 2 full-stack todo application.

## Available Contracts

- `api-contract-v1.yaml` - Version 1 of the API contract defining endpoints, request/response formats, and error handling for the todo application.

## Purpose

These contracts serve as the agreement between frontend and backend teams, ensuring consistent API implementation and enabling parallel development. They also facilitate automated testing and documentation generation.
